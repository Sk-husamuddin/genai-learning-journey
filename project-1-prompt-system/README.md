# Project 1 — Study Assistant Prompt System

## What it does
A prompt system that turns Claude into a 
study assistant — explains any topic from 
simple to deep level with quiz questions.

## How to use
1. Open Claude.ai
2. Paste the system_prompt.txt as system prompt
3. Ask any study topic
4. Get Simple → Medium → Deep explanation + Quiz

## Output Format
- Simple: Beginner analogy
- Medium: Technical explanation  
- Deep: Step by step internal working
- Quiz: 3 questions in JSON format
